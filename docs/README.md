# Documentation

## Development Roadmap

The [Interactive Development Roadmap](roadmap.html) provides a visual overview of all Declutter Assistant features — complete, in-progress, and planned.

**Live version:** [https://thunderdanp.github.io/declutter/roadmap.html](https://thunderdanp.github.io/declutter/roadmap.html)

### How to use

- **Browse** — Filter features by status (Complete, In Progress, Planned) or category (Core, Insurance, Moving, Infrastructure).
- **Edit** — Click any feature card to change its status or add notes. Click the status badge to quick-cycle through statuses.
- **Persist** — Edits are saved to your browser's localStorage automatically.
- **Export/Import** — Use the Export button to download your progress as JSON. Use Import to restore it on another browser or after the page is updated.
- **Reset** — Click "Reset All" to clear saved edits and return to the default state.
